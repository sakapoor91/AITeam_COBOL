# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-22

### Added
- Initial release of EvolutionAI
- Full CardDemo COBOL-to-Java conversion demo (7 modules)
- Six-agent orchestration architecture (Mayor, Analyst, Polecat, Witness, Refinery, Deacon)
- Living metrics dashboard with 4 stakeholder views (Executive, Architecture, Operations, Compliance)
- Reverse engineering reports for all 7 CardDemo modules
- Generated Java/Quarkus code with tests
- OpenAPI 3.1 specifications for all REST endpoints
- Witness review verdicts for all translations
- Observability stack (Langfuse, Prometheus, Grafana)
- Documentation suite (executive summary, methodology guide, best practices, ADRs)
- Docker Compose single-command startup
- CI/CD pipeline with GitHub Actions
- Professional OSS scaffolding (LICENSE, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY)
